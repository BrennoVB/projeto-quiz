# projeto-quiz
🏆 Quiz Esportivo
Um quiz interativo de perguntas sobre esportes, desenvolvido com HTML, CSS e JavaScript puro — sem frameworks ou bibliotecas externas.


🎮 Funcionalidades

10 perguntas sobre esportes variados
Placar em tempo real com número da pergunta e acertos
Tela de resultado final com mensagem personalizada por faixa de pontuação
Botão de "Jogar novamente" para reiniciar sem recarregar a página
Layout responsivo — funciona em desktop e mobile


🛠️ Tecnologias utilizadas

HTML5
CSS3
JavaScript (ES6)


📁 Estrutura do projeto
projeto-quiz/
├── index.html
├── estilo/

    └── script.js

🚀 Como rodar o projeto

Clone o repositório:

git clone https://github.com/BrennoVB/projeto-quiz.git

Acesse a pasta do projeto:

cd projeto-quiz

Abra o arquivo index.html no navegador.


Não é necessário instalar nada — o projeto roda direto no navegador.
